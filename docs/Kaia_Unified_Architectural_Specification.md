## 1. Core Security Axioms & Invariants Reference

Every component of the Kaia architecture must strictly enforce and maintain the following eight non‑negotiable invariants. Any implementation path that introduces a bypass to these rules represents a critical vulnerability.

### 1.1 The Non‑Negotiable Axioms

1. **LLM Outputs are Untrusted Payloads:** All actions, script proposals, parameter assignments, and queries generated by a Large Language Model (LLM) must be handled as untrusted user‑space network input. They must never be allowed direct or unvalidated access to the host shell or privileged APIs.
2. **Deterministic Schema Validation:** All administrative or operations intents must be parsed and strictly validated against strongly typed Pydantic models by a deterministic host process before being passed to an execution layer.
3. **Host‑Retained Execution Control:** The language model operates purely as a reasoning and proposal mechanism. The actual execution layer, policy gatekeeper, and containment boundary remain completely deterministic, hardcoded, and separate from the model’s context window or cognitive processing loop.
4. **Decoupling of Personality and Policy:** Authorization states, privilege assignments, firewall enforcement rules, and token tracking systems must remain completely decoupled from the agent’s conversational personality, affinity tracking matrix, relationship tiers, or active emotional vectors.

### 1.2 System Invariants Matrix

| Invariant ID | Absolute System Constraint | Systemic Failure Mode (If Violated) |
| :--- | :--- | :--- |
| **INV‑001** | The Policy Gate must always **fail closed**. Any socket timeout, network disconnection, parsing exception, or daemon failure results in an absolute, immediate denial of the request. | Privilege escalation via denial‑of‑service or pipeline crashing. |
| **INV‑002** | Structural separation of authorization. Trust verification relies solely on expiring cryptographic capability tokens and explicit system allowlists, never on LLM‑interpreted state or user affinity scores. | Prompt injection manipulating user relationship status to bypass access restrictions. |
| **INV‑003** | Structural database isolation. The security event ledger (`security_events.db`) is strictly append‑only and immutable. The agent holds zero write, update, or delete capabilities over this file. | Post‑exploitation cleanup and log erasure by a compromised agent thread. |
| **INV‑004** | Telemetry sanitization before ingestion. All incoming telemetry streams must pass through character allowlists and hard string truncation fields to eliminate indirect prompt injections hidden in external network metadata. | Arbitrary instruction execution via malicious DNS SNI hostnames or packet payloads. |
| **INV‑005** | Total elimination of shell compilation. The host executor must invoke system binary commands directly via absolute path arrays (subprocess mapping). Shell interpolation, shell strings, or `shell=True` executions are permanently banned. | Command injection via shell metacharacters (`&&`, `||`, `|`, `;`). |
| **INV‑006** | Pure schema‑driven intent. The agent expresses its target actions strictly by generating structured JSON matching defined Pydantic validation contracts; it is completely blocked from passing raw terminal command lines. | Unchecked execution of arbitrary binary parameters. |
| **INV‑007** | Capability Intersection. Effective permissions always resolve via capability intersection ($P_E = P_G \cap P_A$), meaning an individual workspace configuration can never elevate a capability denied by global operator policy. | Compromised code workspace escalating container breakout vectors out‑of‑band. |
| **INV‑008** | Append‑only Audit Ledger isolation. The system transaction log (`audit_ledger.json`) is maintained by a root‑privileged host utility completely outside the directory mapping bounds of the agent’s sandboxed working tree. | Manipulation of the historical administration trail. |

## 3. Out‑of‑Process Typed Policy Gate & Unix IPC Framework

To prevent memory injection exploits and maintain a resilient execution environment, the Policy Gate runs completely out‑of‑process as a root‑privileged standalone systemd daemon communicating over an isolated Unix Domain Socket.

### 3.1 IPC Sockets & Framing Protocol

- **Socket Path:** `/run/kaiacord/policy_gate.sock`  
- **Group Ownership:** `kaiacord` (set via `os.chown` to root user and administrative group bounds)  
- **Permissions Mask:** `0o660` (restricting standard user world‑read/write access)  

**Framing Paradigm:** Every packet starts with a 4‑byte big‑endian unsigned integer indicating the exact byte length of the following UTF‑8 encoded JSON string payload. No newlines or sentinel bytes are used.

```
+---------------------------+-------------------------------------------------+
| Length Prefix (4 Bytes)   | JSON String Payload (Variable UTF‑8 Bytes)     |
| [ Big‑Endian uint32 ]    | {"request_id": "...", "action": "..."}         |
+---------------------------+-------------------------------------------------+
```

## 4. Advanced Local Telemetry Pipeline & Kernel Observers

### 4.1 High‑Performance File Integrity Monitoring (FIM) via `fanotify`

The FIM subsystem interfaces with the Linux kernel’s native `fanotify` API via direct `ctypes` bindings. To eliminate TOCTOU race conditions, the scanning loop extracts the target file descriptor (`fd`) provided by the kernel inside the event metadata and opens `/proc/self/fd/{fd}` for YARA scanning.

**Reference Implementation (from Gemini_Research):**

```python
import os, sys, asyncio, ctypes, struct
from concurrent.futures import ThreadPoolExecutor
from typing import Generator, Dict, Any, List
from pydantic import BaseModel, Field
import yara
import sqlite3
from datetime import datetime

libc = ctypes.CDLL("libc.so.6", use_errno=True)

FAN_CLASS_NOTIF  = 0x00000000
FAN_CLOEXEC      = 0x00000001
FAN_NONBLOCK     = 0x00000002
FAN_MARK_ADD     = 0x00000001
FAN_MARK_MOUNT   = 0x00000010

FAN_MODIFY       = 0x00000002
FAN_CREATE       = 0x00000100
FAN_ATTRIB       = 0x00000004
FAN_ONDIR        = 0x40000000

class FanotifyEventMetadata(ctypes.Structure):
    _fields_ = [
        ("event_len", ctypes.c_uint32),
        ("vers", ctypes.c_uint8),
        ("reserved", ctypes.c_uint8),
        ("metadata_len", ctypes.c_uint16),
        ("mask", ctypes.c_uint64),
        ("fd", ctypes.c_int32),
        ("pid", ctypes.c_int32),
    ]

libc.fanotify_init.argtypes = [ctypes.c_uint, ctypes.c_uint]
libc.fanotify_init.restype = ctypes.c_int
libc.fanotify_mark.argtypes = [ctypes.c_int, ctypes.c_uint, ctypes.c_uint64, ctypes.c_int, ctypes.c_char_p]
libc.fanotify_mark.restype = ctypes.c_int

class YARAMatchMetadata(BaseModel):
    rule: str
    namespace: str
    tags: List[str]
    meta: Dict[str, Any]

class FIMTelemetryPayload(BaseModel):
    timestamp: str = Field(default_factory=lambda: datetime.utcnow().isoformat())
    event_type: str
    pid: int
    process_comm: str
    target_path: str
    yara_matches: List[YARAMatchMetadata] = []

class AppendOnlyAuditTrail:
    def __init__(self, db_path: str = "/var/lib/secdaemon/fim_audit.db"):
        self.db_path = db_path
        self._initialize_database()
    def _initialize_database(self):
        os.makedirs(os.path.dirname(self.db_path), exist_ok=True)
        conn = sqlite3.connect(self.db_path)
        with conn:
            conn.execute("PRAGMA journal_mode=WAL;")
            conn.execute("PRAGMA synchronous=EXTRA;")
            conn.execute("""
                CREATE TABLE IF NOT EXISTS security_log (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    timestamp TEXT NOT NULL,
                    event_type TEXT NOT NULL,
                    pid INTEGER NOT NULL,
                    process_comm TEXT NOT NULL,
                    target_path TEXT NOT NULL,
                    payload_json TEXT NOT NULL
                );
            """)
        conn.close()
    def write_record(self, payload: FIMTelemetryPayload):
        conn = sqlite3.connect(self.db_path)
        with conn:
            conn.execute("""
                INSERT INTO security_log (timestamp, event_type, pid, process_comm, target_path, payload_json)
                VALUES (?, ?, ?, ?, ?, ?);
            """, (payload.timestamp, payload.event_type, payload.pid, payload.process_comm,
                  payload.target_path, payload.model_dump_json()))
        conn.close()

class FIMDaemon:
    def __init__(self, watch_mount: str, rules_path: str):
        self.watch_mount = watch_mount
        self.fan_fd = -1
        self.db = AppendOnlyAuditTrail()
        self.yara_rules = yara.compile(filepath=rules_path)
        self.executor = ThreadPoolExecutor(max_workers=os.cpu_count() or 4)
    def initialize_fim(self):
        self.fan_fd = libc.fanotify_init(FAN_CLASS_NOTIF | FAN_CLOEXEC | FAN_NONBLOCK, os.O_RDONLY)
        if self.fan_fd < 0:
            err = ctypes.get_errno()
            raise OSError(err, f"fanotify_init failed: {os.strerror(err)}")
        mask = FAN_MODIFY | FAN_CREATE | FAN_ATTRIB | FAN_ONDIR
        res = libc.fanotify_mark(self.fan_fd, FAN_MARK_ADD | FAN_MARK_MOUNT, mask, -1, self.watch_mount.encode())
        if res < 0:
            err = ctypes.get_errno()
            os.close(self.fan_fd)
            raise OSError(err, f"fanotify_mark failed: {os.strerror(err)}")
    def _resolve_process_comm(self, pid):
        try:
            with open(f"/proc/{pid}/comm", "r") as f:
                return f.read().strip()
        except FileNotFoundError:
            return "stale_process"
    def _get_fd_filepath(self, fd):
        try:
            return os.readlink(f"/proc/self/fd/{fd}")
        except OSError:
            return "unknown_filepath"
    async def poll_events(self):
        loop = asyncio.get_running_loop()
        while True:
            await self._wait_for_fd()
            data = os.read(self.fan_fd, 4096)
            offset = 0
            struct_size = ctypes.sizeof(FanotifyEventMetadata)
            while offset + struct_size <= len(data):
                metadata = FanotifyEventMetadata.from_buffer_copy(data, offset)
                if metadata.fd >= 0:
                    duplicated_fd = os.dup(metadata.fd)
                    loop.run_in_executor(self.executor, self.process_file_safety, duplicated_fd, metadata.pid, metadata.mask)
                    os.close(metadata.fd)
                offset += metadata.event_len
    async def _wait_for_fd(self):
        loop = asyncio.get_running_loop()
        future = loop.create_future()
        def callback():
            loop.remove_reader(self.fan_fd)
            if not future.done():
                future.set_result(None)
        loop.add_reader(self.fan_fd, callback)
        await future
    def process_file_safety(self, fd, pid, mask):
        if pid == os.getpid():
            os.close(fd); return
        target_path = self._get_fd_filepath(fd)
        process_comm = self._resolve_process_comm(pid)
        event_type = "UNKNOWN"
        if mask & FAN_MODIFY: event_type = "MODIFY"
        elif mask & FAN_CREATE: event_type = "CREATE"
        elif mask & FAN_ATTRIB: event_type = "ATTRIB"
        payload = FIMTelemetryPayload(event_type=event_type, pid=pid, process_comm=process_comm, target_path=target_path)
        try:
            with open(f"/proc/self/fd/{fd}", "rb") as scan_file:
                matches = self.yara_rules.match(data=scan_file.read())
                for match in matches:
                    payload.yara_matches.append(YARAMatchMetadata(
                        rule=match.rule, namespace=match.namespace, tags=match.tags, meta=match.meta
                    ))
        except Exception as e:
            # Fail‑closed: trigger kernel panic on scan failure
            with open("/proc/sysrq-trigger", "wb") as f:
                f.write(b"c")
        finally:
            os.close(fd)
        self.db.write_record(payload)
```

**eBPF Connection Tracker (BCC)** – Reference Implementation:

```python
from bcc import BPF
import socket, struct
from datetime import datetime
from pydantic import IPvAnyAddress

bpf_source_connection_tracker = """
#include <uapi/linux/ptrace.h>
#include <net/sock.h>
#include <linux/socket.h>
struct connection_record_t {
    u32 pid;
    char process_comm[16];
    u32 destination_ip;
    u16 destination_port;
};
BPF_PERF_OUTPUT(socket_connections_buffer);
int trace_socket_connect(struct pt_regs *ctx, struct socket *sock, struct sockaddr *address, int addr_len) {
    if (address->sa_family != AF_INET) return 0;
    struct sockaddr_in *target = (struct sockaddr_in *)address;
    struct connection_record_t record = {};
    record.pid = bpf_get_current_pid_tgid() >> 32;
    bpf_get_current_comm(&record.process_comm, sizeof(record.process_comm));
    record.destination_ip = target->sin_addr.s_addr;
    record.destination_port = be16_to_cpu(target->sin_port);
    socket_connections_buffer.perf_submit(ctx, &record, sizeof(record));
    return 0;
}
"""

class ConnectionTelemetrySchema(BaseModel):
    timestamp: str = Field(default_factory=lambda: datetime.utcnow().isoformat())
    pid: int
    process_comm: str
    destination_ip: IPvAnyAddress
    destination_port: int
    is_beacon_candidate: bool = False

class NetworkTelemetryMonitor:
    def __init__(self):
        self.bpf_context = None
    def begin_monitoring(self, telemetry_callback):
        self.bpf_context = BPF(text=bpf_source_connection_tracker)
        self.bpf_context.attach_kprobe(event="__sys_connect", fn_name="trace_socket_connect")
        def handle_event(cpu, raw_data, size):
            event = self.bpf_context["socket_connections_buffer"].event(raw_data)
            ip_str = socket.inet_ntoa(struct.pack("<I", event.destination_ip))
            payload = ConnectionTelemetrySchema(
                pid=event.pid,
                process_comm=event.process_comm.decode(),
                destination_ip=ip_str,
                destination_port=event.destination_port
            )
            telemetry_callback(payload)
        self.bpf_context["socket_connections_buffer"].open_perf_buffer(handle_event)
    def poll_buffer(self):
        if self.bpf_context:
            self.bpf_context.perf_buffer_poll()
```

### 5.2 Broadcast Protocol Extraction Maps

**Reference Implementation (from Gemini_Research):**

```python
class PassiveNetworkDiscoverer:
    def __init__(self, interface: str):
        self.interface = interface
        self.raw_sock = None
        self.mac_vendor_index = {
            "00:1e:06": "Wintel Corp",
            "3c:a6:16": "Apple Inc.",
            "b4:b5:2f": "Ubiquiti Networks",
            "00:0c:29": "VMware"
        }
    def bind_socket(self):
        self.raw_sock = socket.socket(socket.AF_PACKET, socket.SOCK_RAW, socket.htons(0x0003))
        self.raw_sock.bind((self.interface, 0))
    def _resolve_vendor(self, mac: str) -> str:
        return self.mac_vendor_index.get(mac[:8].lower(), "Unknown Hardware Vendor")
    def read_and_parse_packets(self):
        while True:
            raw_data, _ = self.raw_sock.recvfrom(65535)
            if len(raw_data) < 14: continue
            eth_hdr = struct.unpack("!6s6sH", raw_data[:14])
            src_mac = ":".join(f"{b:02x}" for b in eth_hdr[1])
            proto_type = eth_hdr[2]
            # ARP (0x0806)
            if proto_type == 0x0806:
                arp_raw = raw_data[14:42]
                if len(arp_raw) >= 28:
                    arp_hdr = struct.unpack("!HHBBH6s4s6s4s", arp_raw)
                    operation = arp_hdr[4]
                    sender_ip = socket.inet_ntoa(arp_hdr[6])
                    if operation in (1, 2):
                        yield AssetProfilePayload(
                            mac_address=src_mac,
                            assigned_ip=sender_ip,
                            vendor=self._resolve_vendor(src_mac),
                            detection_vector="ARP",
                            epoch_timestamp=datetime.utcnow().timestamp()
                        )
            # IP (0x0800) – parse UDP for mDNS, LLMNR, SSDP
            elif proto_type == 0x0800:
                ip_raw = raw_data[14:34]
                if len(ip_raw) < 20: continue
                ip_hdr = struct.unpack("!BBHHHBBH4s4s", ip_raw)
                ip_protocol = ip_hdr[6]
                src_ip = socket.inet_ntoa(ip_hdr[8])
                if ip_protocol == 17:  # UDP
                    udp_raw = raw_data[34:42]
                    if len(udp_raw) < 8: continue
                    udp_hdr = struct.unpack("!HHHH", udp_raw)
                    dst_port = udp_hdr[1]
                    if dst_port == 5353:   # mDNS
                        dns_raw = raw_data[42:]
                        hostname = self._extract_dns_hostname(dns_raw)
                        yield AssetProfilePayload(
                            mac_address=src_mac,
                            assigned_ip=src_ip,
                            resolved_hostname=hostname or "Unresolved",
                            vendor=self._resolve_vendor(src_mac),
                            detection_vector="mDNS",
                            epoch_timestamp=datetime.utcnow().timestamp()
                        )
                    elif dst_port == 5355:  # LLMNR
                        yield AssetProfilePayload(
                            mac_address=src_mac,
                            assigned_ip=src_ip,
                            vendor=self._resolve_vendor(src_mac),
                            detection_vector="LLMNR",
                            epoch_timestamp=datetime.utcnow().timestamp()
                        )
                    elif dst_port == 1900:  # SSDP
                        yield AssetProfilePayload(
                            mac_address=src_mac,
                            assigned_ip=src_ip,
                            vendor=self._resolve_vendor(src_mac),
                            detection_vector="SSDP",
                            epoch_timestamp=datetime.utcnow().timestamp()
                        )
    def _extract_dns_hostname(self, payload: bytes) -> str:
        if len(payload) < 12: return ""
        try:
            offset = 12
            labels = []
            while offset < len(payload):
                length = payload[offset]
                if length == 0: break
                if (length & 0xC0) == 0xC0: break  # compressed pointer
                offset += 1
                label = payload[offset:offset+length].decode('utf-8', errors='ignore')
                labels.append(label)
                offset += length
            return ".".join(labels) if labels else ""
        except Exception:
            return ""
```

### 6.1 eBPF System Canary Core

Honey‑tokens are written to static configuration paths (e.g., `/etc/api_keys.json`). The system loads an eBPF tracepoint on `sys_enter_openat` to detect unauthorised access.

**eBPF Program (from Gemini_Research):**

```c
#include <uapi/linux/ptrace.h>
#include <linux/sched.h>
struct canary_alert_t {
    u32 pid;
    u32 uid;
    char process_comm[16];
    char accessed_path[64];
};
BPF_PERF_OUTPUT(canary_telemetry_channel);

int trace_openat_honeytoken(struct pt_regs *ctx, int dfd, const char __user *filename, int flags) {
    char read_path[64] = {};
    bpf_probe_read_user_str(&read_path, sizeof(read_path), filename);
    if (read_path[0] == '/' && read_path[1] == 'e' && read_path[2] == 't' && read_path[3] == 'c' &&
        read_path[4] == '/' && read_path[5] == 'a' && read_path[6] == 'p' && read_path[7] == 'i' &&
        read_path[8] == '_') {
        struct canary_alert_t alert = {};
        alert.pid = bpf_get_current_pid_tgid() >> 32;
        alert.uid = bpf_get_current_uid_gid() & 0xFFFFFFFF;
        bpf_get_current_comm(&alert.process_comm, sizeof(alert.process_comm));
        bpf_probe_read_kernel(&alert.accessed_path, sizeof(alert.accessed_path), read_path);
        canary_telemetry_channel.perf_submit(ctx, &alert, sizeof(alert));
    }
    return 0;
}
```

### 6.2 Network Decoy System & Netlink Firewall Architecture

An isolated network namespace (`ns_decoy_sandbox`) hosts a lightweight mock listener. If a transaction hits a decoy port, the coordinator applies an immediate nftables drop rule via raw Netlink.

**Reference Implementation (from Gemini_Research):**

```python
from pyroute2 import NetNS, NFTables, IPRoute
from pyroute2.nftables.expressions import ipv4addr, verdict

class DecoySystemCoordinator:
    def __init__(self, interface: str, decoy_ip: str = "192.168.100.1"):
        self.interface = interface
        self.decoy_ip = decoy_ip
        self.ns_name = "ns_decoy_sandbox"
        self.nft = NFTables()
        self._initialize_firewall_ruleset()
    def _initialize_firewall_ruleset(self):
        try:
            self.nft.table('add', name='filter_mitigation')
            self.nft.chain('add', table='filter_mitigation', name='input_blocker',
                           hook='input', type='filter', priority=-1)
        except Exception as e:
            sys.stderr.write(f"Unable to initialize firewall: {e}\n")
            sys.exit(1)
    def provision_decoy_isolation(self):
        netns_mgr = NetNS(self.ns_name)
        ip_cmd = IPRoute()
        interface_idx = ip_cmd.link_lookup(ifname=self.interface)[0]
        ip_cmd.link('set', index=interface_idx, net_ns_fd=self.ns_name)
        ns_ip_cmd = IPRoute(netns=self.ns_name)
        ns_ip_cmd.link('set', index=interface_idx, state='up')
        ns_ip_cmd.addr('add', index=interface_idx, address=self.decoy_ip, prefixlen=24)
        ns_ip_cmd.close()
        ip_cmd.close()
    def process_interception(self, attacker_ip: str):
        try:
            self.nft.rule('add', table='filter_mitigation', chain='input_blocker',
                          expressions=(
                              ipv4addr(src=attacker_ip),
                              verdict(code=0)  # DROP
                          ))
        except Exception as e:
            self.lockdown_system_and_panic(f"Firewall state application error: {e}")
    def lockdown_system_and_panic(self, cause: str):
        sys.stderr.write(f"FAIL-CLOSED EXECUTED: {cause}\n")
        with open("/proc/sysrq-trigger", "wb") as f:
            f.write(b"o")  # hard shutdown
```

### 7.3 Ephemeral Verification Sandboxes (`ephem_compile`)

**Reference Implementation (from Gemini_Research):**

```python
import subprocess, tempfile, os

class EphemeralValidator:
    def __init__(self, benign_corpus_dir: str = "/var/lib/secdaemon/benign_test_set"):
        self.benign_corpus_dir = benign_corpus_dir
    def run_sandbox_validation(self, signature_text: str) -> bool:
        with tempfile.NamedTemporaryFile(suffix=".yar", mode="w", delete=False) as f:
            f.write(signature_text)
            temp_rule_path = f.name
        try:
            sandbox_command = [
                "/usr/bin/systemd-run",
                "--quiet", "--wait",
                "--user=false",
                "--property=PrivateTmp=yes",
                "--property=ProtectSystem=strict",
                "--property=ProtectHome=yes",
                "--property=PrivateNetwork=yes",
                "--property=DynamicUser=yes",
                "--property=NoNewPrivileges=yes",
                "--property=CapabilityBoundingSet=",
                "--property=SystemCallFilter=@system-service",
                "--property=RestrictAddressFamilies=none",
                f"--property=BindReadOnlyPaths={temp_rule_path}:{temp_rule_path}",
                f"--property=BindReadOnlyPaths={self.benign_corpus_dir}:/mnt/benign_corpus",
                "/usr/bin/yara",
                temp_rule_path,
                "/mnt/benign_corpus"
            ]
            exec_result = subprocess.run(sandbox_command, capture_output=True, text=True, timeout=10)
            if exec_result.returncode == 0 and not exec_result.stdout.strip():
                return True
            else:
                sys.stderr.write(f"Verification failed: {exec_result.stderr}\n")
        except subprocess.TimeoutExpired:
            sys.stderr.write("Compilation execution exceeded timeout.\n")
        finally:
            if os.path.exists(temp_rule_path):
                os.remove(temp_rule_path)
        return False
```

### 8.2 Token Bucket Telemetry Rate‑Limiting

$$T(t) = \min\left(B, T_{\text{prev}} + r \cdot (t - t_{\text{prev}})\right)$$

- $B$ (Maximum Bucket Burst Capacity) = `100 Tokens`
- $r$ (Token Recovery Generation Rate) = `5 Tokens/Second`

When bucket tokens fall below $1$, incoming logging records are suppressed.
